# Powered by Vercel Branding

All webpages hosted on vercel must contain the "Powered by Vercel" badge, in order to fulfill our sponsorship requirements.

![Powered by Vercel Badge](../../static/img/powered-by-vercel.svg)

Take note of the following **requirements for deployed sites**:

* The badge must link to `https://vercel.com/?utm_source=cse-it&&utm_campaign=oss` on click

* The badge must be visible on all pages

And take note of the following **requirements for the respective Github repo**:

* The project must be open-source (and licensed appropriately)

* The badge must be present in the project's root readme file
